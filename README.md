# Glow Beauty Salon Appointment Booking App

## Project Overview
Glow Beauty Salon is a mobile UI/UX design project that simplifies salon appointment booking through an intuitive and user-friendly interface.

## Features
- Splash Screen
- Login Screen
- Home Screen
- Service Details
- Staff Selection
- Date & Time Selection
- Booking Confirmation
- Success Screen

## Design Goals
- Simple booking flow
- Clean and modern UI
- Easy appointment scheduling
- Improved user experience
